# Disability Awareness Website

## 🌐 Live Website

https://disabilityawareness.netlify.app/

---

## 📌 Project Overview

The **Disability Awareness Website** is designed to educate users about different types of disabilities, promote inclusivity, and provide accessible information for everyone.

The goal of this project is to raise awareness, improve understanding, and support digital accessibility standards.

---

## 🚀 Features

* Informative content about disabilities
* User-friendly and responsive design
* Accessibility-focused layout
* Simple and clean navigation
* Fast-loading static website

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ⚙️ Installation & Setup

To run the project locally:

1. Clone the repository

```
git clone https://github.com/Tswelo23/disabilityAware.git
```

2. Open the project folder

3. Run the project

* Open `index.html` in your browser
